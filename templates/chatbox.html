Chat Box

Download Ollama. You need the desktop application as well as the models. Go to Models and create the phi3:mini model. Give it a name and phi3:mini as the model. Leave system empty for now. It will download the model so let it work it might take a minute. Especially for larger ones. Set the state by clicking the Set State button in the list once the page refreshes. Then come back here to chat box. The model will work a little slow because you are most likely using a CPU instead of a GPU and I have not configured it to work with GPUs yet. I will add support for GPUs soon.
